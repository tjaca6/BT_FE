## Budget tracker app using React, TypeScript and Vite

This is a Budget Tracker that helps you have a better understanding of your spending habits.

## Features
- Add incomes and expenses
- Choose categories 
- See summary: budget, spent, remaining, income
- Tracking spending goals based on categories

## Installation
1. Clone repository

``` git clone https://github.com/tjaca6/BT_FE.git ```


2. Open the folder

``` cd budget_tracker```


3. Install dependencies 

``` npm install ```

4. Start the server 

``` npm run dev```

5. Open local URL shown in terminal, for example 
http://localhost:5173/
